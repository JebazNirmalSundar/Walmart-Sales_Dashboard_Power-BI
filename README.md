# Walmart-Sales_Dashboard_Power-BI
Built a Power BI Sales Dashboard to monitor revenue, profit %, product performance, and payment trends. Applied DAX measures, data modeling, and KPI reporting to transform raw sales data into interactive visual insights supporting business decisions.

## Overview
This dashboard analyzes 10,000+ sales records to track KPIs such as revenue, profit, and regional performance.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Power Query
